JQuery was created in 2006, and very quickly became the most popular JavaScript library in the world.
It was designed to handle Browser Incompatibilities and to simplify HTML DOM Manipulation, Event Handling, Animations, and Ajax.
  • Vanilla JS 5 (2009) pirated adopted much of the jQuery functionality.
  • But it still remained the GoTo tool for AJAX (Asynchronous http(s) Transactions) until the vanilla Fetch API for AJAX was released in 2015 (not part of vanilla JS 6, but about the same time).
  • It was used in most CSS UI libraries such as Bootstrap, which finally went pure vanilla in about 2020 with Bootstrap 5.
  • Note: JQuery is NOT a competitor of ReactJS, which serves a significantly different purpose.
