More Jquery

$() grabs an object. It can be specified by $("#id")
$("#id").html() is a getter
$("#id").html("dog") is a setter
$("div") selects and grabs all of the divs, and
$("div").each(callback) is a callback function, which fires an event for each div
A callback is a function that is bound to an event handler. It is executed when the event is triggered

AJAX

AJAX stands for Asynchronous JavaScript and XML
AJAX requests are made using the XHR (XMLHttpRequest) browser object
The hard part of XHR are the status codes.
JQuery made XHR easier with AJAX.
There is also the fetch() API, which came out in 2017
With AJAX, you can make requests to a server without reloading the page
AJAX takes a JSON object and sends it to the server
The server processes the JSON object and sends a response back to the client
The client processes the response and updates the page
